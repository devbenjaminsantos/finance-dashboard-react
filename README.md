# 💰 Finance Dashboard

Aplicação de controle financeiro pessoal com registro, filtragem e visualização dinâmica de receitas e despesas.

🔗 [Ver projeto ao vivo](https://finance-dashboard-react-six.vercel.app)

## ✨ Funcionalidades

- 📥 Registro de receitas e despesas
- 🗂 Filtragem por categoria
- 📊 Gráficos interativos com Recharts
- 📋 Tabela de transações com filtros
- 💾 Persistência de dados via localStorage
- 🔄 CRUD completo
